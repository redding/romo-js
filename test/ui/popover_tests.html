Popovers

Popovers are UI elements layered in a stack on top of the main body of the page. They auto size and position themselves based on the available screen area and some configuration directives. They can be nested and are automatically closed when interacting with layers below them (e.g. clicking off of a Popover) or hitting the 'Esc' key.

Romo.UI.Modal

Modals are popovers that are opened and placed un-anchored in the viewport. Unlike Dropdowns, Modals can be dragged by users to reposition as needed (e.g. to see the content underneath them, etc). See the examples below for more details and other API settings.

Romo.UI.Dropdown

Dropdowns are popovers that can be toggled open/close and are anchored to the toggle element (by default). They can be anchored (i.e. placed relative to the toggle element) in 4 configurations: bottom-left, bottom-right, top-left, top-right. This setting directs which "corner" of the toggle element to anchor the popover to. See the examples below for more details and other API settings.
